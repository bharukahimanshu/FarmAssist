QuickCrop Routes

Routes

  • GET / - Home page
  • POST /crop-predict - Perform crop prediction
  • POST /crop-predict-temp - Predict crop based on temperature
  • POST /crop-predict-humid - Predict crop based on humidity
  • POST /crop-predict-rain - Predict crop based on rainfall
  • POST /crop-yield-predict - Perform crop yield prediction
  • POST /fertilizer-predict - Perform fertilizer prediction
  • POST /send-result - Send prediction result
  • POST /send-newsletter - Send newsletter to subscribers
  • GET /blogs - Get all blogs
  • POST /add-blog - Add a blog
  • GET /news - Get the news
  • POST /send-message - Send OTP
  • POST /check-subscription - Check subscription status
  • POST /subscribe - Subscribe to the newsletter
  • POST /unsubscribe - Unsubscribe from the newsletter

This page is served by the QuickCrop Flask server.

The frontend is hosted at https://quickcrop.netlify.app/.
